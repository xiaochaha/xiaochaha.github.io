#### 1．利用T-SQL语言编写程序，求1+2!+3!+4!+…..+100!；（10分）
100的阶乘有158位数,不会
算了一下1+2+3+····+100
```sql
DECLARE @i BIGINT,@sum BIGINT
SET @i=1
SET @sum=0
WHILE @i<101 BEGIN
SET	@sum+=@i;
SET	@i=@i+1;
END
PRINT @sum
```
