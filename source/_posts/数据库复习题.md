```sql
--注意：用了综合题的例子!需要修改值
```

#### 1．利用T-SQL语言编写程序，求1+2!+3!+4!+…..+100!；（10分）
```sql
--注意：1+2!+3!+4!+…..+10!
Declare @s int, @i int,@sumi int
Select @s=1,@i=1,@sumi=0
WHILE @i<11 BEGIN
SET @s=@s*@i
SET	@sumi+=@s
SET @i+=1
END
Print @sumi
```
